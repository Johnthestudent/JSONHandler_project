This project is for some block ideas for implementing a Minetest mod or subgame.
Minetest belongs to its respective creators.

The following technologies and languages were used:
- HTML
- JSON
- CSS
- PHP

How to use:

1.) Download the entire project folder into C:/xampp/htdocs folder. In case you don't have xampp, please, install it!

2.) Start xampp with double clicking on xampp-control file.

3.) Start the Apache and the MySQL servers.

4.) Type in your browser: localhost/downloaded_project_name/.

5.) Type in a number between 1 and 10 in the first form. 

6.) Clicking on the Submit button, you will be redirected to the page where you can see the name of the block and its description, depending on this: block id = number you gave.

7.) Should you get to the message saying: block  cannot be found, then go back to the forms section and type in something into the textboxs of the other form. Click the
Add new block button, and then go back to the forms section to click on the Submit button for the n-th number for which previously no block could be shown.
